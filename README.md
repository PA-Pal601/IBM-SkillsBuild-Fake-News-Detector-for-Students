# 📰 Fake News Detector for Students 

## 📌 Project Overview

Fake News Detector is an AI-based system that helps students identify whether a news article is **Real or Fake**.  
The system uses **Natural Language Processing (NLP)** and **Machine Learning algorithms** to analyze news content and detect misinformation.

The project aims to reduce the spread of false information and help students access reliable information.

---

## 🎯 Objective

- Detect fake and real news automatically.
- Analyze news articles using AI techniques.
- Extract important text features using NLP.
- Provide accurate prediction results.
- Help students verify online information.
